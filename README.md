# soot-android
An Example of using Soot + Android

## Getting Started


### Prerequisites
- Android Platform on **USER.HOME**/Android/Sdk/platforms
- JAVA_HOME

### Installing

```shell
git clone https://github.com/jfsr/soot-android.git
```

### Running

```shell
./gradlew run
```
 
### Build eclipse project

```shell
./gradlew clean cleanEclipse eclipse
```
